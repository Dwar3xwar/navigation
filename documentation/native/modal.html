Modal

The Navigation router works with the Modal component from React Native. In our email example, we render a Modal when the user chooses to create a new contact. 

import {Modal} from 'react-native';

<Modal
  visible={this.state.modalVisible}
  onRequestClose={() => this.setState({modalVisible: false})}>
  <Contact />
</Modal>

Navigating in a Modal

var contactNavigator = new StateNavigator([
  {key: 'contact'},
  {key: 'address', trackCrumbTrail: true}
]);

var {contact, address} = contactNavigator.states;
contact.renderScene = () => <Contact />;
address.renderScene = () => <Address />;

contactNavigator.navigate('contact');

<Modal>
  <NavigationHandler stateNavigator={contactNavigator}>
    <NavigationStack />
  </NavigationHandler>
</Modal>

Handling the Android Back Button

import {ModalBackHandler} from 'navigation-react-native';

<ModalBackHandler>
  {handleBack => (
    <Modal
      onRequestClose={() => {
        if (!handleBack())
          this.setState({modalVisible: false});
      }}>
      <NavigationHandler stateNavigator={contactNavigator}>
        <NavigationStack />
      </NavigationHandler>
    </Modal>        
  )}
</ModalBackHandler>
