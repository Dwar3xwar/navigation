Deep Links

A deep link is an external URI that, when clicked, opens a specific scene in your app. After following the setup instructions below, your app is ready to respond to deep link clicks. You create a function inside App.js (outside of any component) that receives the incoming deep link and navigates to the relevant scene. You hook this function up to the Linking API of React Native. In our email example, when a user clicks a link like 'email://app?id=12' we navigate them to the 'mail' scene for that id. Follow the testing instructions below to check that the deep link behaves as expected. 

import {Linking} from 'react-native';

var openLink = (url) => {
  if (url) {
    var id = +url.split('=')[1];
    stateNavigator.navigate('mail', {id});
  }
};
Linking.getInitialURL().then(openLink);
Linking.addEventListener('url', ({url}) => openLink(url));

Setup

Android

Locate the AndroidManifest.xml in the Android project and add: 

<activity
  android:name="com.navigation.reactnative.LinkActivity"
  android:launchMode="singleTask">
    <intent-filter>
    <action android:name="android.intent.action.VIEW" />
    <category android:name="android.intent.category.DEFAULT" />
    <category android:name="android.intent.category.BROWSABLE" />
    <data android:scheme="email" android:host="app" />
  </intent-filter>
</activity>

Locate the MainActiviy.java in the Android project and add: 

import android.net.Uri;
import android.os.Bundle;
import com.facebook.react.bridge.ReactContext;
import com.facebook.react.modules.core.DeviceEventManagerModule;

@Override
protected void onCreate(Bundle savedInstanceState) {
  super.onCreate(savedInstanceState);
  Uri uri = getIntent().getData();
  if (uri != null) {
    ReactContext currentContext = getReactInstanceManager().getCurrentReactContext();
    DeviceEventManagerModule deviceEventManagerModule =
    currentContext.getNativeModule(DeviceEventManagerModule.class);
        deviceEventManagerModule.emitNewIntentReceived(uri);
  }
}

iOS

Locate the AppDelegate.m in the iOS project and add: 

#import <React/RCTLinkingManager.h>

- (BOOL)application:(UIApplication *)application openURL:(NSURL *)url
  sourceApplication:(NSString *)sourceApplication annotation:(id)annotation
{
  return [RCTLinkingManager application:application openURL:url
             sourceApplication:sourceApplication annotation:annotation];
}
