React Native

Rendering Scenes

Like React, the Navigation router works on native apps as well as the web. The Navigation router provides a SceneNavigator component to handle the unique requirements of native app navigation, like managing and animating the stack. You can see it in action the following Twitter example. When you select a tweet, the current view scales back as the new one slides in from the right. When you click the back arrow, the current view slides out to the right as the previous one scales back in.
