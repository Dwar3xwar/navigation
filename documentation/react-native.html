React Native

Rendering Scenes

Like React, the Navigation router works on native apps as well as the web. The Navigation router provides a SceneNavigator component to handle the unique requirements of native app navigation, like managing and animating the stack. You can see it in action the following Twitter example. When you select a tweet, the current view scales back as the new one slides in from the right. When you click the back arrow, the current view slides out to the right as the previous one scales back in.

You create a new StateNavigator passing in an array of States that represent the scenes in your app, just like you would on the web. Make sure you set trackCrumbTrail true to enable the breadcrumb trail stack navigation. 

var stateNavigator = new StateNavigator([
  {key: 'home'},
  {key: 'tweet', trackCrumbTrail: true}
]);

Then you attach a renderScene function to each State. This is similar to the navigated function you'd use on the web except that renderScene returns a component. 

stateNavigator.states.tweet.renderScene = function(data) {
  return <Tweet id={data.id} />;
}
<SceneNavigator
  getUnmountedStyle={() => ({translate: 100, scale: 1})}
  getMountStyle={() => ({translate: spring(0), scale: spring(1)})}
  getMountedStyle={() => ({translate: spring(5), scale: spring(0.9)})}
  getUnmountStyle={() => ({translate: spring(100), scale: 1})}
  animateStyle={({translate, scale}) => ({
    transform: `translate(${translate}%) scale(${scale}, ${scale})`})}
  stateNavigator={stateNavigator} />
