Hyperlinking

Hyperlinks are important. Here's the basis of a good Hyperlink implementation in a Single Page Application: 

// HTML
<a id="anchor" href="some/route" />

// JavaScript
anchor.addEventListener('click', function(e){
    if (!e.ctrlKey && !e.shiftKey) {
        e.preventDefault();
	// Navigate to 'some/route' client side
    }
});

The href attribute makes it act like a Hyperlink while the click listener prevents a trip to the server. If either one of these ingredients is missing then your user experience will suffer.

There are Navigation plugins for Angular, Knockout and React that let you build winning Hyperlinks with one line of UI code. For example, here's how to build a Hyperlink in React that navigates back 1 State along the breadcrumb trail: 

<NavigationBackLink distance={1}>Return</NavigationBackLink>

If you're not using one of these UI frameworks or you're interested in how these plugins are built, then read on.

Each of the three navigating functions on the StateController has a counterpart, with matching parameters, which help to build Hyperlinks: navigate/getNavigationLink, navigateBack/getNavigationBackLink and refresh/getRefreshLink. These counterpart functions return a link which you can pass to the StateController's navigateLink function. For example, the following two code snippets are navigationally identical: 

Navigation.StateController.navigateBack(1);
var link = Navigation.StateController.getNavigationBackLink(1);
Navigation.StateController.navigateLink(link);

Before you can set this returned link as a Hyperlink's href attribute, it needs to be modified by the history manager, for example, the Hash history manager prepends a '#': 

// HTML
var link = Navigation.StateController.getNavigationBackLink(1);
anchor.href = Navigation.settings.historyManager.getHref(link);

// JavaScript
anchor.addEventListener('click', function(e){
    if (!e.ctrlKey && !e.shiftKey) {
        e.preventDefault();
        var link = Navigation.settings.historyManager.getUrl(anchor);
        Navigation.StateController.navigateLink(link);
    }
});
