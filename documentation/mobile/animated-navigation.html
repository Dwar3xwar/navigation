Animated Navigation

Dropping Breadcrumbs

The Navigation router brings native-like navigation to the web. It builds a stack of the scenes a user visits. You can navigate back to any scene in the stack and animate the scenes as they move through and off the stack. The Navigation router builds the stack by dropping a 'breadcrumb' at each visited scene (like the fairy-tale characters Hansel and Gretel). You enable the breadcrumb trail by setting trackCrumbTrail to true against each State. In an email application, we set trackCrumbTrail to true on the 'mail' State so that we can animate out the 'inbox' scene and animate in the 'mail' scene when the user opens an email. 

var stateNavigator = new StateNavigator([
  {key: 'inbox', route: ''},
  {key: 'mail', trackCrumbTrail: true}
]);

Rendering Scenes

stateNavigator.states.inbox.renderScene = function(data) {
  return <Inbox />;
};
stateNavigator.states.mail.renderScene = function(data) {
  return <Mail mailId={data.id} />;
};

Animating

ReactDOM.render(
  <NavigationMotion
    unmountedStyle={{opacity: 0}}
    mountedStyle={{opacity: 1}}
    crumbStyle={{opacity: 0}}
    stateNavigator={stateNavigator}>
    {(style, scene, key) => (
      <div
        key={key}
        style={{opacity: style.opacity}}>
        {scene}
      </div>
    )}
  </NavigationMotion>,
  document.getElementById('app')
);
div {
  position: fixed;
  left: 0;
  right: 0;
  top: 0;
  bottom: 0;
  overflow: auto;
}

Navigating Back

<NavigationBackLink
  distance={1}
  stateNavigator={stateNavigator}>
  Back to Inbox
</NavigationBackLink>
