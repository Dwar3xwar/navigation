Navigation Angular

Navigation Angular is a plugin that lets you build Hyperlinks using Angular's binding syntax. What follows is a series of code snippets taking you through the Angular required for the different types of navigation. Each code snippet begins with a comment to remind you of the JavaScript for the navigation, followed by the equivalent Hyperlink navigation written in Angular.

Navigation to a Dialog

<!-- Navigation.StateController.navigate('inbox'); -->
<a navigation-link="'inbox'"></a>

<!-- Navigation.StateController.navigate('inbox', { pageNumber: 2 }); -->
<a navigation-link="'inbox'" to-data="{ pageNumber: 2 }"></a>

Navigation along a Transition

<!-- Navigation.StateController.navigate('select'); -->
<a navigation-link="'select'"></a>

<!-- Navigation.StateController.navigate('select', { id: 10 }); -->
<a navigation-link="'select'" to-data="{ id: 10 }"></a>

Back Navigation

<!-- Navigation.StateController.navigateBack(1); -->
<a navigation-back-link="1"></a>

Refresh Navigation

<!-- Navigation.StateController.refresh({ pageNumber: 2 }); -->
<a refresh-link="{ pageNumber: 2 }"></a>


<!-- 
var data = Navigation.StateContext.includeCurrentData({ pageNumber: 2 }, ['sort','filter']);
Navigation.StateController.refresh(data); 
-->
<a refresh-link="{ pageNumber: 2 }" current-data-keys="'sort,filter'"></a>


<!-- 
var data = Navigation.StateContext.includeCurrentData({ pageNumber: 2 });
Navigation.StateController.refresh(data); 
-->
<a refresh-link="{ pageNumber: 2 }" include-current-data="true"></a>
