HTML5 History

You change from Hash to HTML5 history by passing in a new HTML5HistoryManager when creating the StateNavigator. If your Url's look like 'http://domain/app/route' then pass the 'app' base path into the HTML5HistoryManager constructor. 

var stateNavigator = new Navigation.StateNavigator([
  {key: 'inbox', route: ''},
  {key: 'mail'},
  {key: 'compose'}
], new Navigation.HTML5HistoryManager());
