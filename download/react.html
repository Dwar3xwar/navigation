Navigation React

This is the page to download Navigation with the React plugin. If you're not using React then you should download Navigation instead.

File

Navigation 1.1.0
Navigation React 1.1.0 
<script src="navigation-1.1.0.min.js"></script>
<script src="navigation.react-1.1.0.min.js"></script>
The non-minified versions of Navigation and Navigation React are also available.

npm

Because navigation is a peer dependency of navigation-react, installing navigation-react isn't guaranteed to install navigation. The safest option is to install navigation and then install navigation-react. 
npm install navigation
npm install navigation-react
var Navigation = require('navigation');
var NavigationReact = require('navigation-react');

Bower

Because navigation is a dependency of navigation-react, installing navigation-react will also install navigation. 
bower install navigation-react
