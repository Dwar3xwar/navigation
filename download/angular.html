Navigation Angular

This is the page to download Navigation with the Angular plugin. If you're not using Angular then you should download Navigation instead.

File

Navigation 1.2.0
Navigation Angular 1.2.0 
<script src="navigation-1.2.0.min.js"></script>
<script src="navigation.angular-1.2.0.min.js"></script>
The non-minified versions of Navigation and Navigation Angular are also available.

npm

Because navigation is a peer dependency of navigation-angular, installing navigation-angular isn't guaranteed to install navigation. The safest option is to install navigation and then install navigation-angular. 
npm install navigation
npm install navigation-angular
var Navigation = require('navigation');
var NavigationAngular = require('navigation-angular');

Bower

Because navigation is a dependency of navigation-angular, installing navigation-angular will also install navigation. 
bower install navigation-angular
