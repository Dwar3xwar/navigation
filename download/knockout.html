Navigation Knockout

This is the page to download Navigation with the Knockout plugin. If you're not using Knockout then you should download Navigation instead.

File

Navigation 1.3.0
Navigation Knockout 1.2.0 
<script src="navigation-1.3.0.min.js"></script>
<script src="navigation.knockout-1.2.0.min.js"></script>
The non-minified versions of Navigation and Navigation Knockout are also available.

npm

Because navigation is a peer dependency of navigation-knockout, installing navigation-knockout isn't guaranteed to install navigation. The safest option is to install navigation and then install navigation-knockout. 
npm install navigation
npm install navigation-knockout
var Navigation = require('navigation');
var NavigationKnockout = require('navigation-knockout');

Bower

Because navigation is a dependency of navigation-knockout, installing navigation-knockout will also install navigation. 
bower install navigation-knockout
