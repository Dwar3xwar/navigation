Downloads

You can start using Navigation today by either downloading the latest version or using a package manager to get at the code. The details are outlined in the first setup step below. Once you've completed this step then, if you're using Angular, Knockout or React, you should follow the additional plugin setup step. This will let you build Hyperlinks in your library's binding syntax.

1. Navigation

Navigation 1.0.0 (minified) 
<script src="navigation-1.0.0.min.js"></script>
Navigation 1.0.0 (non-minified) 
<script src="navigation-1.0.0.js"></script>

npm

npm install navigation
var Navigation = require('navigation');

Navigation.StateInfoConfig.build([...]);

Bower

bower install navigation

npm

The npm install of navigation gives access to all the plugins. 
var NavigationReact = require('navigation/react');
var NavigationBackLink = NavigationReact.NavigationBackLink;

React.createElement(NavigationBackLink, {distance: "1"}, "Go Back")

Bower

The bower install of navigation includes all the plugins as separate files.
