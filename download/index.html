Downloads

Explain that can either get Navigation or the plugin. If you're using Angular, Knockout or React then go straight to the plugin section.

Navigation 1.0.0

The minified version of Navigation. <script src="navigation-1.0.0.min.js"></script> The non-minified version is also available.

Plugins

Navigation Angular 1.0.0

The minified version of Navigation Angular. It includes the Navigation source so you only need to reference this one JavaScript file. <script src="navigation.angular-1.0.0.min.js"></script> The non-minified version is also available.

Navigation Knockout 1.0.0

The minified version of Navigation Knockout. It includes the Navigation source so you only need to reference this one JavaScript file. <script src="navigation.knockout-1.0.0.min.js"></script> The non-minified version is also available.

Navigation React 1.0.0

The minified version of Navigation React. It includes the Navigation source so you only need to reference this one JavaScript file. <script src="navigation.react-1.0.0.min.js"></script> The non-minified version is also available.
