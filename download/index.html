Download

The Navigation router download is made up of two files or packages: Navigation, for the core router, and Navigation React, for the Hyperlink components.

File

Navigation 2.0.0
Navigation React 2.0.2 

<script src="navigation-2.0.0.min.js"></script>
<script src="navigation.react-2.0.2.min.js"></script>
The non-minified versions of Navigation and Navigation React are also available.

npm

npm install navigation navigation-react
var Navigation = require('navigation');
var NavigationReact = require('navigation-react');

Bower

bower install navigation-react
