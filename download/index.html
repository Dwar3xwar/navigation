Navigation Download

This is the page to download Navigation without any plugins. If you're using Navigation with Angular, Knockout or React then you should download Navigation Angular, Navigation Knockout or Navigation React instead.

File

Navigation 1.0.0 
<script src="navigation-1.0.0.min.js"></script>
The non-minified version of Navigation is also available.

npm

npm install navigation
var Navigation = require('navigation');

Bower

bower install navigation
