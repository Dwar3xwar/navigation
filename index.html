Navigation

Navigation is a new kind of router for building Single Page Applications. It works in all browsers and with all web libraries, from Angular to React. Try the interactive tutorial or take a look at the live examples.

TUTORIAL

This interactive tutorial will take you from Navigation zero to Navigation hero in next to no time.

Complete the code exercises in your browser and run them to see if you're right. By the end, you won't look at routing in the same way ever again.

EXAMPLES

See the things you learned from the tutorial put into action in these live example Single Page Applications.

There are examples for Angular, Knockout and React. If you want to build one for your favourite UI library, please get in touch @grahammendick.
