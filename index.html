Navigation

Navigation is a new kind of router. It works in all browsers and with all web libraries, from Angular to React.

Interactive Tutorial Live Examples

Goldilocks

Most routers do either too little or too much. Routers should build Hyperlinks but shouldn't load views. Navigation is the Goldilocks of routers, it does just the right amount.

Breadcrumbs

Navigation tracks the route a user takes, building a dynamic breadcrumb trail. Returning to one of these breadcrumbs recreates the page as the user last saw it.

Plugins

Navigation works with all web libraries. Navigation plugins allow you to build Hyperlinks using your library's UI binding syntax. There are plugins for Angular, Knockout and React.

// Configure the the question and answer States. 
// Map each State to a route. Configure a Transition 
// so you can move from question to answer.
Navigation.StateInfoConfig.build([{
    key: 'quiz', initial: 'question', states: [
        { key: 'question', route: '', 
          defaults: { count: 2 }, transitions: [
            { key: 'next', to: 'answer' }
        ]},
        { key: 'answer', route: 'thanks/{count}' }
    ]
}]);
// Navigate forward from question to answer, passing
// the number selected.
next.onclick = function (e) {
    var data = { count: things.value };
    Navigation.StateController.navigate('next', data);
}
// Navigate backward from answer to question.
goBack.onclick = function (e) {
    Navigation.StateController.navigateBack(1);
}
var dialogs = Navigation.StateInfoConfig.dialogs;
var quiz = dialogs.quiz;
// When in the question State show the question.
quiz.states.question.navigated = function () {
    question.style.display = 'block';
}
// When not in the question State hide the question.
quiz.states.question.dispose = function () {
    question.style.display = 'none';
}
// When in the answer State show the answer.
quiz.states.answer.navigated = function (data) {
    answer.style.display = 'block';
    count.textContent = data.count;
}
// When not in the answer State hide the answer.
quiz.states.answer.dispose = function () {
    answer.style.display = 'none';
}
// Tell Navigation you're ready to go.
Navigation.start();
