Navigation

A NEW KIND OF ROUTER THAT WORKS IN ALL BROWSERS AND WITH ALL WEB LIBRARIES

Interactive Tutorial Live Examples

Hyperlinks

Navigation works with all web libraries. Navigation plugins allow you to build Hyperlinks using your library's UI binding syntax. There are plugins for Angular, Knockout and React.

Breadcrumbs

Navigation tracks the journey a user takes, building a dynamic breadcrumb trail. Returning to one of these breadcrumbs recreates the page as the user last saw it.

Isomorphic

Navigation works just as well on the server as it does on the client. Use it with React and Node to write SEO-friendly apps with fast initial load times.

Navigating Forward and Back

Thanks! I'm glad you have things.

Try out navigating by running the example above. Press the 'Next' button to navigate forward, then press the 'Back' button to navigate back. Have a look at the JavaScript and HTML for this example.

Navigation is a State machine. You configure States and the movement between them. Each State represents a route. You attach methods to States that get called when that State is entered or exited.

Navigation keeps the Url in step as you navigate and pass data between States. Navigation builds a breadcrumb trail by tracking visited States. You can navigate back along this trail.

Learn more by completing the interactive tutorial. 


// Configure the the question and answer States. 
// Map each State to a route. Configure a Transition 
// so you can move from question to answer.
Navigation.StateInfoConfig.build([{
    key: 'quiz', initial: 'question', states: [
        { key: 'question', route: '', transitions: [
            { key: 'next', to: 'answer' }
        ]},
        { key: 'answer', route: 'thanks/{count}' }
    ]
}]);
// Navigate forward from question to answer, passing
// the number selected.
next.onclick = function (e) {
    var data = { count: things.value };
    Navigation.StateController.navigate('next', data);
}
// Navigate back from answer to question.
goBack.onclick = function (e) {
    Navigation.StateController.navigateBack(1);
}
var dialogs = Navigation.StateInfoConfig.dialogs;
var quiz = dialogs.quiz;
// When in the question State, show the question.
quiz.states.question.navigated = function () {
    question.style.display = 'block';
}
// When not in the question State, hide the question.
quiz.states.question.dispose = function () {
    question.style.display = 'none';
}
// When in the answer State, show the answer.
quiz.states.answer.navigated = function (data) {
    answer.style.display = 'block';
    count.textContent = data.count;
}
// When not in the answer State, hide the answer.
quiz.states.answer.dispose = function () {
    answer.style.display = 'none';
}
// Tell Navigation you're ready to go.
Navigation.start();
