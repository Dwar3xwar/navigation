Exercise 5: Configuring a Transition

When a user selects a person from the grid you need to move them to the details view. You need to navigate from the listing State to the details State. To navigate between States within a Dialog you use a Transition. You create a Transition as part of the configuration, adding it below the State you want to move from and specifying the State to move to. In this Exercise, you'll create the Transition before navigating along it in the next one.
